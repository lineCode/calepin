Note: mp3’s won’t loop smoothly. It seems that if a frame of the mp3 is not full, it gets padded. This padding causes a very noticeble gap in playback. Therefore smooth looping without some sort of gapless playback support from the [encoder and decoder is impossible](https://www.rockbox.org/tracker/11891)
