## Gmail filter HTML attributes

Filtered:

- `class`
- `id`
- `data-*`
- `contenteditable`
- `contextmenu`
- `draggable`
- `hidden`
- `itemprop`
- `spellcheck`
- `style`
- `tabindex`

Kept:

- `dir`
- `lang`
- `title`
- `alt`
- `role`
- `aria-labelledby`
